Template Element

It’s a native HTML element where you can write a snippet of HTML that you later want to insert dynamically. The browser doesn’t render that first.
    In JavaScript, you grab the template using getElementById, clone its content, fill that clone HTML node with whatever text you need, and finally append the element to the DOM with appendChild. It's a bit like JSX if you know that, but natively. Why is this better? It’s safer than innerHTML, no XSS vulnerabilities It’s more performant when adding multiple elements because innerHTML forces the browser to relay out everything you modify, while appendChild only affects the newly added DOM element You get proper HTML structure and syntax highlighting It’s genuinely elegant
