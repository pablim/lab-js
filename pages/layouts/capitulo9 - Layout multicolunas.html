Layout multicolunas

Atualmente o navegador Opera suporta nativamente a maioria das funcionalidades desse módulo. Os navegadores Firefox, Chrome e Safari suportam com uso dos prefixos -moz- e -webkit-

As proprieadades CSS para criar e estiliza elementos multicolunas são listadas a seguir:

  • column-count
  • column-width
  • columns
  • column-gap
  • column-rule
  • column-span
  • column-fill
  • break-before
  • break-after
  • break-inside

column-count

Essa propriedade se destina a definir o número de colunas a ser criado em um elemento. Os valores possíveis para ess propriedade são: auto e inteiro. O valor padrão é auto e significa que não há divisão em colunas ou que a divisão é definida por outra propriedade (por exemplo: column-width). O valor inteiro é um número inteiro maior que zero, definindo a quantidade de boxes colunas a serem inseridos no elemento multicolunas.

.multi-colunas {
 -moz-column-count: 4;
 -webkit-column-count: 4;
 column-count: 4;
 padding: 0 10px;
 border: 1px solid;
}

column-width

Essa propriedade define a largura das colunas a serem ciradas em um elemento. Os valores possíves para essa propriedade são: auto ou uma medida CSS. O valor padrão é auto e sigifica que não há divisão em colunas ou que a divisão é definida por outra propriedade (por exemplo: column-count). O valor medida CSS deve ser maior do que zero, definindo a largura dos boxes colunas a serem inseridos no elemento multicolunas.

Ao contrário da propriedade column-count, para qual o número de colunas é fixo, essa propriedade faz com que o número de colunas seja dinâmico, ou seja, a quantidade de columnas necessárias para acomodar o conteúdo depende da resolução.

.multi-colunas {
 -moz-column-width: 150px;
 -webkit-column-width: 150px;
 column-width: 150px;
}

columns

Essa propriedade é a forma abreviada de se definir column-count e column-width. .multi-colunas {
 -moz-columns: 4 150px;
 -webkit-columns: 4 150px;
 columns: 4 150px;
}

O primeiro valor é referente a propriedade column-count e o segundo valor é referente a propriedade column-width. A forma abreviada admite a declaração de dois valores ou de um valor, e, nesse caso, o valor omitido é considerado como auto. Nenhum navegador atual suporta essa propriedade abreviada.

column-gap

Essa propriedade se destina a definir a largura das colunas a serem criadas em um elemento. Os valores possíveis para essa propriedade são: normal ou uma medida CSS. O valor padrão é normal e significa que o espaçamento entre colunas será calculado pelo navegador. O valor recomendado pela especificação para a largura do espaçamento entre colunas é de 1em. O valor medida CSS deve ser maior do que zero.

.multi-colunas {
 -moz-column-count: 4;
 -webkit-column-count: 4;
 column-count: 4;

 -moz-column-gap: 60px;
 -webkit-column-gap: 60px;
 column-gap: 60px;
}

column-span

Essa propriedade se destina a fazer com que um elemento contido no box multicoluna se estenda por toda a largura do box, como se não houvesse colunas no sentido horizontal do elemento. Os valores possíveis para essa propriedade são: none eall. O valor padrão é none e faz com que o elemento permaneça confiando na sua coluna. O valor all estende o elemento por todas as colunas.

Essa propriedade é suportada nativamente pelo navegador Opera e pelos navegadores Safari e Chrome, com uso do prefixo -webkit-.

.multi-colunas {
 -moz-column-count: 4;
 -webkit-column-count: 4;
 column-count: 4;
}

.multi-colunas2 {
 -moz-column-count: 4;
 -webkit-column-count: 4;
 column-count: 4;
}

h4 {
 text-align: center;
 background: #ccc;
 margin: 0;
}

.multi-colunas2 h4 {
 -moz-column-span: all;
 -webkit-column-span: all;
 column-span: all;
}

column-fill

Essa propriedade se destina a definir como o conteúdo será distribuído pelas colunas. Os valores possíveis para essa propriedade são: auto e balance. O valor padrão é auto e faz com que o conteúdo preencha as colunas sequencialmente. O valor balance faz com que o agente de usuário tente, se for possível, distribuir o conteúdo igualmente pelas colunas.

Essa propriedade atualmente não é suportada por nenhum navegador, e sua sintaxe é conforme mostrado a seguir:

.multi-colunas {
 column-fill: balance;
}

break-*

break-before e break-after

Essas propriedades se destinam a controlar a quebra de colunas/páginas, tendo como referência o ponto imediatamente antes (break-before) ou depois (break-after) do elemento para a qual foi definida. Os valores possíveis e seus significados são:

Valor Descrição
auto Não há controle de quebra. Esse pe o valor padrão da propriedade.
always Força quebra de página antes (break-before) ou depois (break-after) do elemento
avoid Evita quebra de coluna/página antes (break-before) ou depois (break-after) do elemento
left Força a quebra de uma ou duas páginas antes (break-before) ou depois (break-after) do elemento e faz com que a próxima página seja formatada como página esquerda.
right Força a quebra de uma ou duas páginas antes (break-before) ou depois (break-after) do elemento e faz com que a próxima página seja formatada como página direita.
page Força quebra de página antes (break-before) ou depois (break-after) do elemento.
column Força quebra de coluna antes (break-before) ou depois (break-after) do elemento.
avoid-page Evita quebra de página antes (break-before) ou depois (break-after) do elemento.
avoid-column Evita quebra de coluna antes (break-before) ou depois (break-after) do elemento.

Note que algumas propriedades se aplicam a colunas e páginas, outras somente a colunas e outras ainda somente a páginas.

break-inside

Essa propriedade se destina a controlar a quebra de colunas/páginas, tendo como referência o conteúdo do elemento para o qual foi definido. Os valores possíveis e seus significados são:

Valor Descrição
auto Não há controle de quebra. Esse pe o valor padrão da propriedade.
avoid Evita quebra de coluna/página dentro do conteúdo do elemento.
avoid-page Evita quebra de página dentro do conteúdo do elemento.
avoid-column Evita quebra de coluna dentro do conteúdo do elemento.

Note que algumas propriedades se aplicam a colunas e páginas, outras somente a colunas e outras ainda somente a páginas.

As propriedades break-before, break-after e break-inside atualmente são suportadas somente pelo navegador Opera.

 -moz-column-rule: 3px double #000;
 -webkit-column-rule: 3px double #000;
 column-rule: 3px double #000;
 text-align: justify;

Overflow de coluna

É perfeitamente possível ocorrer o caso em que um elemento multicolunas tenha uma quantidade de colunas cuja soma das larguras seja maior do que a largura do elemento. Nesse caso, as colunas excedentes e seus respectivos conteúdos ultrapassarão o limite direito de elemento. As colunas além dequele limite são chamadas de colunas em overflow.

O aparecimento de colunas em overflow poderá ocorrer em um dos seguintes casos:

  • quando se declara uma altura fixa para o elemento coluna (uso das declarações heigth ou max-heigth);
  • em determinadas situações, quando se declara break_* para elementos nas colunas;
  • em determinadas situações, quando a largura da página tem dimensões limitadas.

.multi-colunas {
max-height: 160px;
-moz-column-count: 3;
-webkit-column-count: 3;
column-count: 3;

-moz-column-gap: 18px;
-webkit-column-gap: 18px;
column-gap: 18px;

-moz-column-rule: 3px double #000;
-webkit-column-rule: 3px double #000;
column-rule: 3px double #000;

text-align: justify;
}
